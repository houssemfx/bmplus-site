# Meubles BM+ — Site Vitrine

Clean, modern showcase site for **Meubles BM Plus (BM+)**, a custom furniture maker based in Kolea, Algeria.

Built with **React + Vite + Framer Motion**. Refined luxury-minimal aesthetic — warm ivory, deep ink, brass/wood accents — with video clips of the collection and WhatsApp-first CTAs.

## Features
- Full-screen looping video hero
- Asymmetric collection grid (video + photo) with hover motion
- Savoir-faire / values / stats sections
- WhatsApp devis CTAs (`0781 41 81 42`) + floating contact button
- Fully responsive, French copy
